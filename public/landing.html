Offer薪资速算器
谈薪不拍脑袋

一个纯前端工具型网页，帮助求职者在谈薪过程中快速计算、对比和决策不同薪资方案。

🚀 立即体验 📂 GitHub 仓库

核心功能

5 个模块，覆盖谈薪前中后的关键计算场景

🎯

期望方案反推

基于当前待遇和期望涨幅，自动反推目标月Base，谈薪报价更有底气。

📝

Offer 汇总录入

录入实际 Offer 的月薪、薪数、公积金比例等，自动计算年包总包。

⚖️

9 项指标对比

期望方案与 Offer 逐项对比，年包、税后、公积金差距一目了然。

📊

年包速查表

月Base 25K~40K × 12~18薪 完整对照表，快速定位自己处在哪一档。

🎨

动态背景切换

支持丝绸流场、雨落寒窗、纯色背景切换，长时间使用也不枯燥。

🔒

纯前端隐私安全

所有计算在浏览器本地完成，薪资数据不上传服务器，隐私更安心。

技术栈

现代前端工具链，轻量、快速、易维护

React 19 TypeScript Vite 8 Tailwind CSS 3.4 Radix UI
