## 1. 项目概述

**项目名称**：Offer薪资速算器（salary-negotiator）

**一句话定位**：一个纯前端、数据不上传的求职薪资计算与对比工具，帮助求职者在谈薪前算清不同 Offer 的真实收益。

**在线地址**：https://g1968712236-create.github.io/salary-negotiator/

**源码仓库**：https://github.com/g1968712236-create/salary-negotiator

## 2. 需求背景

求职者在拿到 Offer 后，往往面临以下问题：

- **数字太多，算不过来**：月薪、年终奖、公积金、社保、绩效、补贴、期权、签字费等每一项都影响真实收入，手动计算容易漏算。
- **怕报高被刷，报低吃亏**：不知道自己的目标月 Base 该开多少，谈薪时心里没底。
- **Offer 一多就纠结**：A 公司月薪高但薪数少，B 公司底薪低但公积金高，到底哪个更划算？
- **对税后收入没有体感**：HR 报的是年包，扣完个税、社保、公积金后到手可能差一截。
- **隐私顾虑**：薪资信息敏感，用户不希望把数据上传到第三方服务器。

现有的通用计算器无法同时处理「多 Offer 对比 + 公积金 + 社保五险 + 个税 + 专项扣除 + 固定支出」的组合场景，因此需要一个专门面向谈薪决策的工具。

---

## 3. 产品定位

面向**求职者**和**HR/猎头**的轻量级薪资决策辅助工具。

- **工具属性**：纯前端网页，打开即用，无需注册、无需下载 App。
- **安全属性**：所有计算在浏览器本地完成，数据仅保存于 localStorage，不上传服务器。
- **决策属性**：不仅算年包，更算税后收入、净收入、公积金总额、社保成本，帮助用户做出可量化的 Offer 选择。

## 4. 目标用户与用户画像

### 4.1 核心用户

| 用户类型 | 特征 | 核心诉求 |
|---|---|---|
| **在职求职者** | 已有 1~3 年工作经验，正在看外部机会 | 对比当前待遇与多个 Offer，判断跳槽是否划算、涨幅是否合理 |
| **应届生 / 转行者** | 对薪资结构（Base、薪数、公积金、社保）概念不熟悉 | 了解月 Base、薪数、公积金、社保、个税等概念对年包的影响 |
| **HR / 猎头** | 需要快速为客户或候选人测算不同方案 | 快速测算年包、成本与税后收入，辅助沟通 |

### 4.2 典型用户画像

**画像 A：小李，互联网产品经理，28 岁**
- 当前月薪 28K，15 薪，想跳槽。
- 拿到两个 Offer：A 公司 32K×14，B 公司 30K×16，公积金比例不同。
- 痛点：粗略看 A 月薪高，但算完年包和公积金后发现 B 更划算，自己算不清。
- 使用场景：把两个 Offer 都录入工具，生成对比图发给家人一起参谋。

**画像 B：小王，后端开发，25 岁**
- 第一次跳槽，不知道怎么和 HR 谈期望薪资。
- 希望涨 30%，但不知道这个目标对应的月 Base 应该是多少。
- 使用场景：输入当前待遇和期望涨幅，让工具反推出目标月 Base，作为谈薪锚点。

**画像 C：陈猎头，35 岁**
- 经常要给候选人解释不同公司方案差异。
- 痛点：口头解释不够直观，容易扯皮。
- 使用场景：用工具生成 PNG 对比图，直接发给候选人。

## 6. 核心功能

| 功能模块 | 说明 | 对应文件 |
|---|---|---|
| **方案管理** | 统一维护「当前岗位 / 期望 / 多个 Offer」方案，支持新增、删除、重命名、切换编辑 | `src/App.tsx` |
| **期望反推** | 基于当前待遇和期望涨幅，自动反推目标月 Base | `src/domain/salary.ts` |
| **待遇综合对比** | 年包、税后收入、净收入、公积金、社保等 9+ 维度横向对比，柱状图直观展示差距 | `src/components/MultiOfferDiffView.tsx`、`src/components/SalaryChart.tsx` |
| **年包速查表** | 自定义月 Base 范围，生成 12~18 薪完整对照表 | `src/components/SalaryTable.tsx` |
| **个税与年终奖税率表** | 内置工资薪金综合所得与年终奖单独计税税率表，可自定义修改 | `src/domain/tax.ts`、`src/components/TaxBracketsEditor.tsx` |
| **导出 PNG 对比图** | 一键生成待遇对比报告图片，方便保存与分享 | `src/components/ExportReport.tsx` |
| **动态背景切换** | 丝绸流场、雨落寒窗、纯色背景随心换 | `src/components/Background.tsx` |

## 7. 价值主张

- **算得清**：把年包、税后、净收入、公积金、社保全部纳入计算口径。
- **比得明**：多 Offer 9 项指标横向对比，涨幅用红绿色直观标注。
- **推得准**：输入期望涨幅即可反推目标月 Base，谈薪报价有依据。
- **用得安**：纯前端运行，数据不上传，薪资隐私不泄露。
- **传得便**：一键导出 PNG 对比图，适合分享给他人参考。

---

## 8. 典型使用场景

1. **谈薪前**：输入当前待遇和期望涨幅，反推出合理的目标月 Base。
2. **收 Offer 后**：把当前岗位和新 Offer 录入工具，对比真实收益。
3. **多 Offer 纠结时**：一次性录入多个方案，用柱状图和表格快速决策。
4. **面试准备**：用年包速查表快速定位自己的薪资档位。
5. **帮朋友参谋**：导出 PNG 对比图，直观展示不同方案的差距。
