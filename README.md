## ✨ 核心功能

| 模块 | 说明 |
|------|------|
| 🎯 **期望方案反推** | 基于当前待遇和期望涨幅，自动反推目标月Base |
| 📝 **Offer 汇总录入** | 录入月薪、薪数、公积金比例，自动计算年包 |
| ⚖️ **9 项指标对比** | 期望 vs Offer 逐项对比，差距一目了然 |
| 📊 **年包速查表** | 月Base 25K~40K × 12~18薪 完整对照 |
| 🎨 **动态背景切换** | 丝绸流场、雨落寒窗、纯色背景随心换 |
| 🔒 **纯前端隐私安全** | 数据不上传，计算在浏览器本地完成 |

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/g1968712236-create/salary-negotiator.git
cd salary-negotiator

# 安装依赖
npm install

# 本地开发
npm run dev

# 构建生产版本
npm run build
```

构建产物位于 `dist/` 目录，为纯静态文件，可直接部署到 GitHub Pages、Vercel、Netlify 等平台。

## 📦 部署到 GitHub Pages

本项目已配置 GitHub Actions 工作流，推送代码到 `main` 分支后会自动构建并部署。

如需手动启用：

1. 进入仓库 **Settings → Pages**
2. **Source** 选择 **GitHub Actions**
3. 推送代码到 `main` 分支
